NAME

SyntheticPlaybackEngine – Generate and send synthetic events to system event managers

SYNOPSIS

        SyntheticPlaybackEngine.py -u URL [OPTION] TEMPLATES

DESCRIPTION

SyntheticPlaybackEngine simulates multiple computers, generating synthetic system events and sending them to an event manager via HTTP POST. The events are generated from randomly selected templates located in the TEMPLATES directory; a single template will be used if TEMPLATES points to a template file. By default, the playback engine simulates one host, creating one series of synthetic events and sending the events as fast as possible.

        REQUIRED

        -u, --url URL

                The URL of the system event manager to send event data to.

        OPTIONAL

        -i, --iterations NUM

Run through NUM template files. Specifying 0 will run continuously; press Ctrl-C to quit. Default setting is 1.

        -m, --machines NUM

The number of machine hosts to simulate -- a thread will be spawned for each machine. Default setting is 1.

        -n, --rate NUM

Limit the rate of events sent over the network to NUM events per second. Default setting is 0 (unlimited).

        -o, --output FILE

Write the generated events out to FILE.

        -v, --version

                Print the current version of SyntheticPlaybackEngine

EXAMPLES

To generate events from one random template and send the events to a local manager:

        python SyntheticPlaybackEngine.py –u https://127.0.0.1:9443 templates/

To generate events from a single template and send the events to a local manager:

        python SyntheticPlaybackEngine.py -u https://127.0.0.1:9443 sample.txt

To simulate 3 hosts and send synthetic events at 400 events per second:

python SyntheticPlaybackEngine.py -u https://127.0.0.1:9443

-m 3 -n 400 templates/

To simulate 10 hosts and generate events indefinitely, while limiting network throughput to 1000 events per second:

                python SyntheticPlaybackEngine.py -u https://127.0.0.1:9443

-m 10 -i 0 -n 1000 templates/
